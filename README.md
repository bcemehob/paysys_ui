# Wallet Maintaining Helper app UI

## Onboarding Process

### Prerequisite

* NodeJs 11+
* Npm

### Installation

Go to the root directory and run from shell:

```
$ npm i
```
### Running

Install & run backend application, then run from shell:
```
$ npm run dev
```
Server will start on http://localhost:3000
